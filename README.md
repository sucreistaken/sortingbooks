# sortingbooks
Book isminde bir sınıf tasarlayınız. Bu sınıf Comparable interface'den kalıtım alıp "compareTo" metodunu override ediniz. Bu metodun içinde kitabı A'dan Z'ye isme göre sıralayan kodu yazınız. Bu sınıftan 5 tane nesne oluşturun ve nesneleri Set tipinde bir yapısında saklayınız. Sonra ikinci kez Set tipinden bir veri yapısı kullanın ve kitapları sayfa sayısına göre sıralamasını sağlayınız.
Book sınıfı kitap ismi, sayfa sayısı, yazarın ismi, yayın tarihi değişkenlerinden oluşmaktadır.

Design a class named "Book" that inherits from the "Comparable" interface and overrides the "compareTo" method. Write the code inside this method to sort the books by name in alphabetical order. Create five instances of this class and store them in a data structure of type Set. Then, use a second data structure of type Set and sort the books by the number of pages.

The Book class consists of variables for book title, number of pages, author's name, and publication date.
